# Simple MML to Standard MIDI File converter

This tool is a part of in-development synthesis.js.

## Setup

$ npm install -g mml2smf

## MML

command|mml
-------|---
notes|c, d, e, f, g, a, b, +, -
octave|o[octave], &lt;, &gt;
note length|l[note length]
gate time|q[1-8]
tempo|t[tempo]
next track|;
tie|^

## Development

To launch:

$ npm start

To build:

$ npm run build
